## 使用方法

1. 下载[本程序](https://github.com/Enaium/BiligameAddictionNotLimited/releases)
2. 安装运行启动(最低安卓7)
3. 使用安卓7及以下版本(可以用模拟器)或者已经root的设备
4. 打开WIFI设置代理(Host和Port和启用时一致,同一网络环境将Host设为192.168.xx.xx)
5. 保存证书安装
